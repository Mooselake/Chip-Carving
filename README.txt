CNC chip carving utilties

rosette_generator.html is a simple gemini created app to create a swirl rosette svg file suitable for use with VCarve or similar software with parameters for the number of petals and direction.  Each created element is a closed path and is separated from adjacent elements by a configurable gap spacing to reduce chipping on sharp edges

twirl_rosette_generator.html is a revised version using AI studio with minor changes
